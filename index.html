lite-embeds

Privacy-first facade web components for social embeds.

Zero third-party JS until the user clicks. Under 3 KB gzipped per package.

GitHub npm
Try it: open DevTools → Network tab → hard reload. You'll see zero requests to twitter.com, spotify.com, player.vimeo.com, instagram.com, or tiktok.com. Click any embed below to inflate it on demand.
