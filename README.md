# lite-embeds-demo

Live demo for [`@lite-embeds/*`](https://github.com/abderrahimghazali/lite-embeds) — a family of privacy-first facade web components for social embeds.

**Demo:** https://lite-embeds-demo.vercel.app/

## Develop

```bash
npm install
npm run dev
```

The demo consumes the published npm packages (`@lite-embeds/twitter`, `@lite-embeds/spotify`, `@lite-embeds/vimeo`, `@lite-embeds/instagram`, `@lite-embeds/tiktok`).

## Deploy

Vercel auto-detects Vite — no `vercel.json` needed. Push to `main` and import on Vercel.
